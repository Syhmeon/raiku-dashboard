📚 Data Sources & Methodology
Rewards & Fees: Dune Analytics query 6773409 — solana.rewards (Staking, Voting, Fee, Rent) + Jito tip accounts
Commission & Validators: Dune query 6773227 — stake-weighted commission from solana.rewards.commission field
Active Stake: Dune query 6776267 — SUM(pre_balance) of Staking rewards = total delegated SOL earning rewards
Fee Breakdown: Dune query 6776270 — solana.transactions + vote_transactions, base = signatures × 5000 lamports
Price & FDV: CoinGecko API (last 365 days), Dune-derived for older epochs
Burns: Pre-SIMD96 (epoch <620): 50% of all fees burned. Post-SIMD96: 50% of base fees only (priority fees 100% to leader)
APY formula: APY = reward × (365.25 / epoch_duration_days) / active_stake
Active Stake measures delegated SOL that actually earned rewards (excludes stake on delinquent validators)
